## System Requirements

### Minimal
* OS: Windows 10 (Update 1607+) / Windows 8.1 / Windows 7 SP1
* RAM: 256 MB
* .NET Framework 4.7.2

### Recommended
* OS: Windows 10 (Latest Update)
* RAM: 2 GB
* .NET Framework 4.7.2
* Beefy PC for item icons and search highlight
